# Job Portal Clone

A minimal job portal website similar to Naukri.com built using only HTML (no CSS or JavaScript).

## Features

- **Homepage** - Job search form and featured jobs
- **Job Listings** - Search and filter jobs by location, experience, salary
- **Job Details** - Complete job descriptions with application forms
- **User Registration** - Comprehensive signup for job seekers
- **User Login** - Authentication for existing users
- **Job Posting** - Employers can post new job openings
- **User Dashboard** - Track applications and view recommendations

## Pages Included

1. `index.html` - Homepage
2. `jobs.html` - Job listings with search filters
3. `job-details.html` - Detailed job view with application form
4. `register.html` - User registration form
5. `login.html` - User login page
6. `post-job.html` - Job posting form for employers
7. `dashboard.html` - User dashboard
8. `application-success.html` - Application confirmation page

## How to Use

1. Open `index.html` in any web browser
2. Navigate through the portal using the navigation menu
3. All forms are functional (though backend processing would need to be implemented)

## Technologies Used

- HTML5 only (semantic markup)
- Form validation using HTML attributes
- No external dependencies
